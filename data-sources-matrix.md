# 데이터 소스 및 정책 매트릭스 (dogcamp)

## 1. 요약 원칙

- 우선 순위:
  1. 공공데이터 / 지자체 캠핑장
  2. 소규모 캠핑 포털 / 개별 캠핑장 사이트
  3. 대형 상업 포털(여행/예약 플랫폼)은
     - 공식 API / 제휴가 가능한 경우에만 데이터 연동
     - 그 외에는 HTML 크롤링 대신 딥링크 중심 전략

- 공통 원칙:
  - 각 사이트의 `robots.txt` 및 이용약관을 준수
  - 크롤링이 명확히 금지된 경우, HTML 크롤링은 하지 않음
  - 가능하면 공식 API / 공공 데이터셋을 우선 사용

## 3. 1차 PoC 대상 제안

- 1차 PoC 우선 후보:
  - **한국관광공사 고캠핑 (공공 API 기반)**  
  - **캠핑맵 (약관/robots 확인 후 HTML 크롤링)**

- 역할 분담:
  - 고캠핑: 기본 캠핑장 데이터(위치, 시설, 반려견 가능 여부 등)의 “골격” 제공
  - 캠핑맵: 공공 데이터에 없는 일부 정보(사진/설명/편의시설 등) 보완용 후보

- 대형 포털(다음 캠핑 등):
  - 당장은 **제휴 또는 공식 API 없이는 HTML 크롤링 대상에 포함하지 않음**
  - 대신 필터/검색 UX, 반려견 관련 표기 방식 등을 벤치마킹하는 용도로 활용
